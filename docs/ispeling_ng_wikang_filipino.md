## I. Ang Alfabetong Filipino

Ang alfabetong Filipino ay binubuo ng **28 letra** na ganito ang ayos:

A, B, C, D, E, F, G, H, I, J, K, L, M, N, Ñ, NG, O, P, Q, R, S, T, U, V, W, X, Y, Z

Sa 28-letrang ito ng alfabeto:

- **20 letra** ang nasa dating ABAKADA  
  *(A, B, K, D, E, G, H, I, L, M, N, NG, O, P, R, S, T, U, W, Y)*
- **8 letra** ang dagdag  
  *(C, F, J, Ñ, Q, V, X, Z)*

### Ang Ngalan ng mga Letra

Ang tawag sa mga letra ng alfabetong Filipino ay ayon sa tawag-Ingles, maliban sa **Ñ (enye)** na tawag-Kastila.

## II. Mga Tuntuning Panlahat sa Ispeling o Pagbaybay

### A. Ang Pasalitang Pagbaybay

**Paletra** ang pasalitang pagbaybay sa Filipino—isa-isang pagbigkas ng mga letra ng salita, pantig, daglat, akronim, inisyal, at simbolo.

#### Salita

- boto — /bi-o-ti-o/
- plano — /pi-el-ey-en-o/
- Fajardo — /kapital ef-ey-jey-ey-ar-di-o/
- vinta — /vi-ay-en-ti-ey/
- jihad — /jey-ay-eich-ey-di/

#### Pantig

- it — /ay-ti/
- kon — /key-o-en/
- trans — /ti-ar-ey-en-es/
- pa — /pi-ey/
- tsart — /ti-es-ey-ar-ti/

#### Akronim

- MERALCO — /em-i-ar-ey-el-si-o/
- LEDCO — /el-i-di-si-o/
- PANDAYLIPI — /pi-ey-en-di-ey-way-el-ay-pi-ay/
- ARMM — /ey-ar-em-em/
- ASEAN — /ey-es-i-ey-en/

#### Daglat

- Bb. — /kapital bi-bi/
- G. — /kapital ji/
- Gng. — /kapital ji-ay-en-ey-en-ji/
- Kgg. — /kapital key-ji-ji/
- Dr. — /kapital di-ar/

#### Inisyal ng Tao

- MLQ — /em-el-kyu/
- CPR — /si-pi-ar/
- JVP — /jey-vi-pi/
- LKS — /el-key-es/
- AGA — /ey-ji-ey/

#### Inisyal ng Samahan / Institusyon

- KWF — /key-dobolyu-ef/
- PSLF — /pi-es-el-ef/
- KBP — /key-bi-pi/
- PLM — /pi-el-em/
- MSU — /em-es-yu/
- NGO — /en-ji-o/

#### Simbolong Pang-agham at Pang-Matematika

- Fe — /ef-i/
- H₂O — /eich-tu-o/
- NaCl — /en-ey-si-el/
- lb. — /el-bi/
- kg. — /key-ji/
- v — /vi/

### B. Ang Pasulat na Pagbaybay

Manatili ang **isa-sa-isang tumbasan ng tunog at letra** sa pasulat na pagbaybay, ngunit pinaluluwag ang gamit ng walong dagdag na letra.

## III. Ang Panghihiram

Isang realidad ang panghihiram ng Filipino mula sa Ingles, Kastila, at iba pang wika.

### Mga Lapit sa Panghihiram

1. **Gamitin ang umiiral na leksikon**
   - attitude → saloobin
   - rule → tuntunin
   - ability → kakayahan
   - wholesale → pakyawan
   - west → kanluran

2. **Kumuha mula sa katutubong wika**
   - hegemony → *gahum* (Cebuano)
   - imagery → *haraya* (Tagalog)
   - husband → *bana* (Hiligaynon)
   - Muslim priest → *imam* (Tausug)

3. **Bigkasin at baybayin sa Filipino**
   - cheque → tseke
   - liquido → likido
   - education → edukasyon
   - radical → radikal
   - coup d’état → kudeta
   - kimono → kimono

## IV. Mga Tiyak na Tuntunin sa Gamit ng Walong Letra

### Letrang C

- Panatilihin: *calculus, cellphone*
- Palitan ng **S** kung /s/: *central → sentral*
- Palitan ng **K** kung /k/: *cake → keyk*

### Letrang Q

- Panatilihin: *quantum*
- Q → KW: *quarter → kwarter*
- Q → K: *quota → kota*

### Letrang Ñ

- Panatilihin: *Santo Niño*
- Ñ → NY: *piña → pinya*

### Letrang X

- Panatilihin: *x-ray*
- X → KS: *exam → eksam*

### Letrang F

- *foto, fokus, futbol*

### Letrang J

- *jam, dyaket, sabjek*

### Letrang V

- *video, valyu, varayti*

### Letrang Z

- *zoo, zebra, bazaar*

## V. Mga Diptonggo

- ia → ya / iya
- ie → ye / iye
- io → yo / iyo
- ua → wa / uwa
- ue → we / uwe
- ui → wi / uwi

## VI. Mga Digrapo

### CH

- Panatilihin: *chips, Chavez*
- CH → TS: *checklist → tseklist*

### SH

- Panatilihin: *shampoo*
- SH → SY: *shooting → syuting*

### NG

Ginagamit para sa tunog /ng/ sa lahat ng posisyon.

## VII. Ang Pantig at Palapantigan

### Kayarian ng Pantig

- P: *u-pa*
- KP: *ma-li*
- PK: *is-da*
- KPK: *han-da*
- KKPKK: *trans-port*

## VIII. Ang Pag-uulit ng Pantig

- Patinig → patinig lang ang inuulit
  - *alis → a-a-lis*
- KP → inuulit ang KP
  - *basa → babasa*
- KK → dalawang paraan
  - *prito → piprito / priprito*

## IX. Ang Gamit ng Gitling

Ginagamit ang gitling sa:

- pag-uulit: *araw-araw*
- unlapi + patinig: *mag-alis*
- pinagsamang salita: *bahay-aliwan*
- unlapi + tanging ngalan: *maka-Diyos*
- ika- + numero: *ika-3*
- apelyidong pinagsama: *Macapagal-Arroyo*
- paghahati ng salita sa dulo ng linya
